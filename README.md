# 🌐 Sitio web (Frontend) desarrollado con HTML5 y CSS3

Este proyecto corresponde a la **primera evaluación sumativa de Frontend**.  
El objetivo fue aplicar lo aprendido en **HTML y CSS**, además de practicar el uso de **Git y GitHub** para subir un proyecto.

---

## 🚀 Desarrollo de la actividad

1. Crear un repositorio en GitHub con el formato:  
   `Nombre_Apellido_Eva1`  
   Ejemplo: `Luis_Padilla_Eva1`

2. Crear una carpeta con el mismo nombre del repositorio e inicializar el proyecto con **Git**:
   - Configurar Git
   - Agregar archivos al *stage*
   - Realizar *commit* de los archivos
   - Subir el proyecto a GitHub mediante comandos

3. Desarrollar una página web con:
   - **HTML** para la estructura
   - **CSS** para los estilos
   - Comentarios en ambos archivos identificando las partes de la estructura

4. Evidencia:  
   - Capturas de pantalla completas mostrando el uso de Git y GitHub.  
   - Si el *print* no es completo, se considera incorrecto.

---

## 📖 Tema de la página

La página aborda el tema: **Educación Cívica en Chile**.  
Debe contener:
- Texto informativo
- Imágenes
- Enlaces (links)
- Estilos aplicados con CSS

---

## 🎯 Objetivos de aprendizaje

1. Aplicar conocimientos básicos de **HTML y CSS**.  
2. Utilizar **Git** como controlador de versiones.  
3. Subir proyectos a **GitHub** correctamente.  
4. Documentar el trabajo con evidencia visual.  

---

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **CSS3**

---

## ⚙️ Ejecución del proyecto

1. Abrir **Visual Studio Code**.  
2. Abrir la carpeta del proyecto.  
3. Ejecutar con **F5** y seleccionar un navegador.  
4. Visualizar la página web.
